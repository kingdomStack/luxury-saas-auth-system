## Overview

SecureAuth is a production-ready PHP authentication system that pairs enterprise-grade security with a premium, SaaS-inspired UI. Built for developers who want polished, conversion-ready login flows without compromising on backend security best practices.

## ✨ Features

- 🔒 **Bcrypt password hashing** — industry-standard credential security
- 🛡️ **CSRF protection** — token-based request validation
- 🔑 **Session hardening** — secure, tamper-resistant session handling
- 🎨 **Luxury SaaS-style UI** — clean, modern, conversion-focused design
- 📱 **Fully responsive** — optimized across desktop and mobile
- ⚙️ **Zero-config friendly** — runs on XAMPP with MySQL out of the box

## 🛠️ Tech Stack

- **Backend:** PHP (MVC pattern)
- **Database:** MySQL
- **Environment:** XAMPP
- **Security:** PDO prepared statements, bcrypt, CSRF tokens

## 🚀 Getting Started

```bash
git clone https://github.com/kingdomStack/luxury-saas-auth-system.git
```

1. Move the project folder into your `htdocs` directory
2. Import the provided `.sql` file into MySQL via phpMyAdmin
3. Configure your database credentials in `config.php`
4. Launch via `http://localhost/secureauth`
